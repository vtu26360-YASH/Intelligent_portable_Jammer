# Intelligent_portable_Jammer
Overview

The Intelligent Portable Jammer Using AI and IoT is an AI-powered wireless monitoring and adaptive signal management system designed to detect and analyze unauthorized wireless communication activities in restricted environments. The project focuses on monitoring Wi-Fi, Bluetooth, and mobile communication signals using Software Defined Radio (SDR), Artificial Intelligence, and IoT technologies.

The system continuously scans nearby RF signals, processes the collected data using machine learning algorithms, and provides real-time monitoring through an IoT dashboard. The project is designed with portability, low latency, and adaptive signal analysis capabilities for secure environments such as examination halls, secure facilities, and restricted zones.

Real-time Wi-Fi and Bluetooth signal detection
AI-based wireless signal classification

Project Workflow
RF signals are captured using RTL-SDR and Wi-Fi adapters.
Signal data is processed using GNU Radio and FFT analysis.
Features such as RSSI, packet activity, and frequency patterns are extracted.
Machine learning algorithms classify authorized and unauthorized activities.
Real-time monitoring and alerts are displayed through the IoT dashboard.
